Инструкция по настроке расширение RecentTracks для Twitch TV

Для того что бы расширение работало необходимо настроить сервис LastFM. LastFM может получать текущую музыку из аудио плеерах как Youtube, Spotify, Google Play Music, Deezer, SoundCloud, Mixcloud, Yandex Music и еще много других.

Начальная инструкция

  1. Необходимо создать аккаунт LastFM. Ссылка для регистрации https://www.last.fm/ru/join

  2. Необходимо подключить LastFM с музыкальным плеером на которой вы слушаете музыку. Подробнее можете узнать по ссылка: https://www.last.fm/about/trackmymusic

  3. Теперь необходимо настроить расширение просто добавив в настройках логин аккаунта LastFM в поле Логин пользователя LastFM.

  4. Для начало необходимо создать API account для интеграции. По ссылке: https://www.last.fm/api/account/create. После созданий аккаунта вам выдадут API key. API key необходимо вставить в настройках расширения, в поле API KEY LastFM.

  5. Сохраните настройки и все готово.
